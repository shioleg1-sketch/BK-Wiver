# Статус проекта BK-Wiver

## 📊 Обзор проекта

### Проект
**BK-Wiver** - система удаленного доступа с единым desktop-приложением и серверной частью.

### Цель
Предоставить удаленный доступ к рабочему столу с высокой производительностью и качеством изображения.

## 🎯 Целевые показатели

### Текущие показатели
- **FPS:** ~20-25 кадров/сек
- **Качество изображения:** HD/SD
- **CPU usage:** 60-80%
- **Bitrate:** 2-4 Mbps

### Целевые показатели
- **FPS:** 30-40 кадров/сек ✅
- **Качество изображения:** HD/4K ✅
